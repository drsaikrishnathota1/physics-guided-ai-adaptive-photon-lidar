\section*{Data and Code Availability}

The simulation source code, experiment configurations, validation tests,
environment records, and processed numerical results are maintained in a
version-controlled research repository. The code release associated with
the submitted manuscript will identify the exact source commit used to
produce the reported results.

The study uses synthetically generated photon-count observations produced
by the documented optical forward model. No personal, clinical, classified,
or proprietary operational data were used.
