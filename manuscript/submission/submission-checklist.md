# Short Communication Submission Checklist

## Scientific Accuracy

- [ ] Every parameter matches the implemented source code.
- [ ] Every numerical result matches summary.json.
- [ ] The policy equation matches the implemented scoring rule.
- [ ] “Information gain” is qualified as heuristic or information-inspired.
- [ ] No artificial-intelligence claim remains.
- [ ] No quantum-advantage claim appears.
- [ ] False-focus rate is not described as target-absent false-alarm rate.
- [ ] Simulation-only limitations are clearly stated.

## Manuscript

- [ ] Title is consistent throughout all files.
- [ ] Abstract contains no unsupported claims.
- [ ] Introduction clearly identifies the research gap.
- [ ] Methodology contains complete reproducible equations.
- [ ] Results report absolute and relative improvements accurately.
- [ ] Discussion separates findings from interpretation.
- [ ] Limitations are explicit.
- [ ] Conclusion does not overstate generalizability.

## References

- [ ] Every BibTeX entry has been verified.
- [ ] Every cited work appears in references.bib.
- [ ] Every reference is cited in the manuscript.
- [ ] No invented volume, issue, page, year, or DOI remains.
- [ ] Recent single-photon LiDAR literature is represented.
- [ ] Optics & Laser Technology literature is represented where relevant.

## Figures and Tables

- [ ] Architecture diagram is completed.
- [ ] Physics-validation figure is completed.
- [ ] Detection comparison figure is completed.
- [ ] Photon-efficiency figure is completed.
- [ ] All axes include units.
- [ ] All figure fonts are readable.
- [ ] Figures are cited in numerical order.
- [ ] Table parameters match the source audit.
- [ ] Results table matches saved output.

## Submission Files

- [ ] Main manuscript PDF.
- [ ] Editable LaTeX source.
- [ ] Bibliography file.
- [ ] Separate figure files.
- [ ] Highlights file.
- [ ] Cover letter.
- [ ] Graphical abstract, if requested.
- [ ] Declaration of competing interests.
- [ ] Data and code availability statement.
- [ ] Author contribution statement.
